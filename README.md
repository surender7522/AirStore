# AIR-STORE
### A simple object store on your pc, fast & scalable

# Requirements
- Python
- Poetry (rest is installed by this)

#Getting Started
```shell
poetry shell
chmod +x run.sh
./run.sh
```

Now navigate to http://localhost:8000/docs to use the store
